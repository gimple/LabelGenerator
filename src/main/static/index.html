Label Generator {{version}}

Loading the directory

You can easily generate your own label with your name on it!

To do so just load the following route: 

          GET /label/your_name.png:

label should be on of the following five options

  • owner_only
  • instructed
  • documented
  • public
  • give_away

your_name field should be the text, you want to write on the label. In general the text can be arbitrary long, but the fontsize will decrease for longer texts.

Format: Beside .png we also provide .jpeg files. Just change the ending of the route.

For example, if you need an instructed label for your name "Max Mustermann", you query /instructed/Max Mustermann.png. I did already prepare this for you: try it!

Loading spaceapi

To load a space use 

        GET /label/your_name:
The :label: variable is the urlencoded version of the name in the directory. E.g. %2Fdev%2Ftal.

Example: /spaces/%2Fdev%2Ftal
Load stats

Load 

        GET /stats/:spacename:.html
to show a stats site.

Example: stats/%2Fdev%2Ftal.html

Example: stats/%2Fdev%2Ftal.html
